HOW TO USE THIS INDICATOR IN TRADINGVIEW:

1. Unzip this package and open the `indicator.pine` file in any text editor (like Notepad, VS Code).
2. Copy the entire contents of `indicator.pine`.
3. Open TradingView (https://www.tradingview.com) and load any chart.
4. Look at the bottom of the chart panel and click on the "Pine Editor" tab.
5. Paste the copied code into the editor (overwrite any default placeholder text).
6. Click "Save" at the top right of the Pine Editor. Give it a name (e.g., "Predictor Dashboard").
7. Click "Add to chart" right next to the Save button.
8. The EMA lines, BUY/SELL labels, and the Side Dashboard Panel with real-time technical metrics, volume surge status, and market insights will render instantly overlayed on the chart!
